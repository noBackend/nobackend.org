Come, dream code with us!

Forget all the constraints of today's browsers. Then write down the code of your dreams for all the tasks you need to build your app. This is dreamcode. Here's another example:

User Accounts

Imagine the typical backend tasks for user authentication would exist right in the browser.

Data Store

Imagine saving and finding user data would just work, persistent and synchronised across devices.

Emails

Imagine you could send emails with JavaScript, multipart, with attachments?

Sharing

Imagine you could share user data by providing read or write access to specific users.

File Conversions

Imagine you could convert all kind of things to all kind of different things, e.g. taking a screenshot of a dom element or converting another website to a screenshot

Payments

Imagine you could accept purchases with pure JavaScript in the browser.

Make Coffee. Or Tea.

Imagine you could control other devices with a simple JavaScript API, right from your browser.
